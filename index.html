Enchanted Matchcraft

Moonlit Relics

Swap spellbound orbs, trigger cascades, and fill the kingdom treasury.

How to Play

Tap or click one orb, then a neighboring orb to swap them. Match 3 or more to collect relics.

Build cascades for bonus score. Four and five matches create stronger runes.

Controls

Mouse / Touch: select and swap adjacent relics

R: restart run

F: toggle fullscreen

App Store Path

This prototype is mobile-first and can later be wrapped for iPhone/iPad distribution.
